# Notebooks
## Cinematic Emotion Lab

This directory contains all Jupyter notebooks organized by type.

## Structure

```
notebooks/
├── exploratory/      # EDA, hypothesis testing, preliminary analysis
├── experiments/      # Formal numbered experiments (NB-EXP-XXX)
└── visualizations/   # Standalone visual analysis notebooks
```

## Naming Convention

```
NB-[TYPE]-[XXX]_[topic-slug]_[YYYY-MM-DD].ipynb
```

| Prefix | Type |
|--------|------|
| `NB-EXP` | Formal experiment |
| `NB-EDA` | Exploratory analysis |
| `NB-VIZ` | Visualization |
