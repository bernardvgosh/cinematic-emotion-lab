## 4. Public Data Release Terms

The public corpus subset released with v0.2 (acoustic feature vectors, anonymized
cue identifiers, structural metadata) is made available for academic research and
educational use under the following terms:

**Permitted:**
- Analysis of the public feature subset for academic research
- Use in course materials or educational contexts with attribution
- Inclusion in comparative studies with other MIR datasets, with clear attribution
- Publication of findings derived from the public subset, with citation

**Not permitted:**
- Commercial use of any kind without prior written permission
- Distribution of the public subset outside the repository without attribution
- Presentation of the public subset as your own data collection
- Use of the public subset in a manner that implies access to embargoed labels or findings

**Attribution requirement:**
Any academic use of the public corpus subset must cite this repository using the
citation in CITATION.cff and reference docs/citation-and-use-policy.md.

## 5. The Composer Gap — Conceptual Framework

The "Composer Gap" — the conceptual framing of composer-machine interpretive divergence
as a primary research object rather than an error metric — is an original theoretical
contribution by Bernard G. first documented in this repository.

Academic work that builds on, extends, critiques, or references this conceptual
framework should cite this repository and, upon publication, the associated peer-reviewed
paper. Derivative research that substantially adopts the Composer Gap framing without
citation would constitute a failure of academic attribution norms.
